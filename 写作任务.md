## 定时任务设置方法

在 Claude Code 里输入以下命令（任选其一）：

```
/loop 10m 开始《鉴宝天师》写作任务——读取写作任务.md，然后按八步流程执行
/loop 1h 开始《鉴宝天师》写作任务——读取写作任务.md，然后按八步流程执行
```

**注意**：每次开新 Claude Code 窗口需要重新设一次定时任务。

【第一步】读全部10个文件

【第三步】写作
【第四步】字数检查（≥2000字）
【第五步】近5章剧情一致性交叉检查
【第六步】新设定冲突检查
【第七步】同步更新所有文件
【第八步】汇报
