## 【卷一主要场景】

### 聚宝斋（典当行）

| 项目 | 内容 |
|------|------|
| 位置 | 老城区巷子里，门脸不大 |
| 外观 | 褪色木质牌匾，"聚宝斋"三字苍劲有力（邱远山手书），门槛高，青砖墙 |

| 光线 | 白天偏暗，灯泡昏黄 |
| 氛围 | 老旧、局促，透着没落 |
| 首次出现 | ch01 |
| 备注 | 邱远山时代的辉煌已不在 |

|------|------|

### 青龙山废弃矿区

| 项目 | 内容 |
|------|------|
| 位置 | 青龙山北麓，距江城老城区约40公里 |
| 外观 | 八十年代开采有色金属留下的遗迹，废弃于1991年；矿洞纵横交错，像掏空了内脏的巨兽趴伏在山脊上 |
| 13号矿洞 | 矿区内最深的主矿洞；入口被铁栅栏封了（现已被林远剪断）；洞壁初期粗糙有凿痕，渐变为光滑的原生岩（成因不明）；木质支撑梁部分腐烂 |
| 矿眼（矿洞最深处） | 矿洞尽头120米深处有人工封闭的混凝土墙，墙中心画着红漆圆圈和十字（与邱远山玉佩纹路一致）；混凝土墙后约3米是石门——遗迹真正入口 |
| 石门（符文之门） | 矿眼最深处；表面刻有先秦以前风格的符文（与玉佩纹路完全一致）；门缝透出微弱白光；门后有比B级器灵更高等级的存在 |
| 灵性环境 | 矿洞深处岩壁有微弱灵性渗透（背景值比正常高0.3%）；凌晨2-4点/6:20-6:40脉冲时段，符文之门释放强烈灵性 |
| 气味 | 潮湿、霉味、浓重金属气息 |
| 光线 | 人工照明（强光手电） |
| 氛围 | 幽闭、压抑、远古；矿洞越深处越不像人造，暗示超自然 |
| 首次出现 | ch38 |
| 备注 | 1987年发现先秦以前上古遗迹；父亲邱远山在此守护24年；林远1991年封闭此处 |

## 使用原则

1. 每章写作前，检查本章涉及的场景是否已有卡片
2. 场景外观以本卡为准，正文不得私自改动
3. 若剧情需要在场景内添加新陈设，在"备注"栏说明
4. 新场景在首次出现章节写完后，补充到本文件
5. 同一场景在后续章节出现，描写基调与本卡保持一致（细节可适度调整）
