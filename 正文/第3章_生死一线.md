# 第三章 生死一线

事情的爆发比邱辰想象中来得更快。

第三天中午，他正在后院整理一批刚收来的旧货，李长河气急败坏地从外面冲进来。

"王老板！出大事了！"

邱辰透过窗户往外看，只见一辆黑色面包车停在门口，车上下来三个人——为首的是个光头壮汉，脖子上纹着一条张牙舞爪的蜈蚣，后面跟着两个马仔，一脸凶相。

是"四海贷"的人。

邱辰心里咯噔一下。

四海贷是江城有名的民间高利贷，利息按天算，逾期不还就上门泼油漆、堵锁眼，手段狠辣。他爸邱远山当年就是借了他们的钱，才落得个倾家荡产。

"谁是邱辰？！"

光头大汉一脚踹开门，冲着店里吼了一嗓子。

王德发正在柜台后面打算盘，被吓得一哆嗦，笔都掉地上了。

"这位兄弟，"王德发堆起笑脸迎上去，"小店是做正经生意的，不知道——"

"少跟我扯淡！"光头大汉一把推开他，"邱远山的儿子是不是在这儿干活？给我滚出来！"

店里鸦雀无声。

李长河缩在角落，连大气都不敢出。

邱辰站在后院门口，两条腿像灌了铅一样沉重。

他知道这一天迟早会来，但没想到会这么快。

他深吸一口气，走了出去。

"我就是。"

光头大汉上下打量了他一眼，从口袋里掏出一张皱巴巴的欠条，上面写得清清楚楚——邱远山，八万三千元，借款人按的手印。

"你爸五年前借的钱，现在本金加利息，一共三十二万。"光头把欠条拍在他脸上，"今天要么拿钱，要么拿命。"

邱辰握着那张欠条，指节发白。

三十二万。

他现在身上还剩两百块。

"我没钱。"他抬起头，直视着光头。

"没钱？"光头冷笑一声，抬手一挥，"那就别怪我不客气了。把他带走，让他去夜总会坐几天台，看看他妹妹的学费能不能凑出来。"

两个马仔立刻扑上来，扭住了邱辰的胳膊。

"等等！"王德发突然开口了，脸上带着一种说不清的表情，"这位大哥，邱辰是我们店的员工，他的事我管不了。但他爸当年欠的钱，跟我没关系。我这店可是正经生意，您几位要是想闹事，我只能报警了。"

光头斜了他一眼："报警？行啊，你报。我倒要看看，是警察来得快，还是你店里这些东西烧得快。"

王德发脸色一变，嘴巴张了张，到底什么都没说出来。

他靠不住。

邱辰早就知道。

他被两个马仔架着往外拖，嘴里还喊着："放开我！王叔，帮我报个警——"

"报什么警？"李长河的声音从角落里飘出来，阴恻恻的，"我们什么都没看见。"

邱辰被拖出门口，后背撞在面包车冰凉的车身上。

他挣扎着回头看了一眼聚宝斋的门脸——褪色的"聚宝斋"三个字在正午的阳光下显得格外刺眼。

门已经关上了。

没有人帮他。

没有人。

就在两个马仔把他往车里塞的时候，他的手腕突然碰到了挂在脖子上的一根红绳。

红绳的另一端，系着那枚墨绿色的玉佩。

一瞬间，玉佩像是活过来了一样——滚烫得几乎要把他的皮肤烫穿，一股巨大的力量从胸口炸开，顺着他的手臂蔓延到全身。

"嘶——！"

他忍不住倒吸一口凉气。

那是一种他从未体验过的感觉。像是有什么东西在他体内撕开了一道口子，又像是有一道光强行灌进了他的身体。

两个马仔被他突然的反应吓了一跳，下意识松了手。

邱辰单膝跪在地上，浑身剧烈颤抖，额头上青筋暴起，豆大的汗珠顺着脸颊滚落。

"你他妈的装什么死？"光头大汉皱起眉头，走过来要拽他。

然而就在他的手指碰到邱辰肩膀的一瞬间，一股无形的力量将他震退了三步！

光头大汉跌坐在地上，满脸惊骇："这……这是什么玩意儿？！"

邱辰跪在地上，眼睛紧闭，牙关紧咬。

他能感觉到体内有什么东西正在苏醒——像是一台尘封多年的机器突然通上了电，各个部件开始自行运转。

然后，他听到了一个声音。

不是从耳朵里传来的，而是直接出现在他的意识里，像是一道冰冷的电子合成音：

**「全能职业系统启动中……」**
**「宿主生命体征异常，系统强制绑定……」**
**「绑定完成。欢迎宿主邱辰。」**

邱辰猛地睁开眼睛。

他看到的世界变了。

面包车还是面包车，光头还是光头，但他的视野左上角突然出现了一块半透明的光屏，上面密密麻麻显示着各种他看不懂的数据：

```
【全能职业系统 v1.0】
宿主：邱辰
综合评级：D-
可用技能点：0
声望值：0
当前职业：未激活
系统商城：已解锁
主线任务：待领取
日常任务：待刷新
```

与此同时，一个蓝色的进度条在他视野中央浮现：

**「新手引导任务进行中……」**

光头大汉从地上爬起来，脸色铁青，冲两个马仔吼道："给我把他拽上车！管他什么妖魔鬼怪，今天不还钱谁都别想好过！"

两个马仔再次扑上来。

但这一次，邱辰没有反抗。

他只是缓缓站了起来，抬起头，看向光头的眼睛。

那一刻，光头莫名感觉到一股彻骨的寒意从脊背窜上来。

"你们，"邱辰的声音很轻，却清晰得可怕，"给我听好了。"

"三十二万，我会还。但不是现在。"

"你们要是敢动我妹一根头发，我会让你们知道什么叫后悔。"

光头被他盯得头皮发麻，下意识退了一步。

"走！"他冲两个马仔使了个眼色，三人连滚带爬地上了面包车，一溜烟地开走了。

邱辰站在原地，看着面包车的尾灯消失在街角。

他低头，把那枚滚烫的玉佩从胸口扯了出来。

玉佩的温度正在慢慢恢复正常，但上面的纹路似乎比之前更深了一些——像是有什么东西被唤醒了。

他再次看向视野左上角的光屏。

光屏上跳出了一行新的文字：

**「新手引导完成。系统奖励已发放。」**
**「获得被动技能：【基础鉴真】D级——30cm范围内，可鉴别古董真伪与基本断代。」**
**「获得技能点：1点。」**
**「主线任务已解锁：潘家园淘宝。请在24小时内完成第一次古董捡漏。」**
**「任务奖励：激活第一个职业——鉴宝师D+。」**

邱辰攥着那枚玉佩，站在七月的烈日下。

他突然想笑。

——他妈的，人生还能再离谱一点吗？

但他没有笑。

因为他知道，不管这个系统是幻觉还是真的，从这一刻起，他的人生已经没有退路了。

他转过身，看了看不远处的聚宝斋。

想了想，他转身往另一个方向走去。

潘家园。

既然系统要他在那儿捡漏，那就去。

反正他现在身上只有两百块和一枚来历不明的玉佩。

还有一条命。

（本章完）
