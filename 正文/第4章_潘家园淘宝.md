# 第四章 潘家园淘宝

邱辰从聚宝斋出来的时候，太阳已经偏西了。

他站在巷子口，低头看了看手里的两百块钱。

纸币皱巴巴的，被汗水浸得有些发软。他把它们仔细叠好，塞进牛仔裤的贴袋里，用手压了压，确认塞得结实。

两百块。

这是他现在全部的身家。

他转身，往老城区的公交站走去。潘家园在城市的另一头，坐公交要倒一趟车，全程大概四十分钟。

他在心里盘算着。

两百块的本金，在潘家园能干什么？

潘家园这地方，邱辰从小就跟老爸去过无数次。那里的规矩他门儿清——地摊上的东西，开口价至少是成交价的五倍甚至十倍，真正的买家得会还价，还得有一双能看穿真假的眼睛。

没有本钱不要紧。只要东西够好，他可以先拿下，再找下家转手——这叫"空手套白狼"，古董圈里叫"借鸡生蛋"。

前提是，他得先看得准。

邱辰下意识攥了攥胸口。

红绳系着的玉佩贴在皮肤上，微微发凉。自从下午那次觉醒之后，玉佩就安静了下来，既不发热也不发冷。但他知道，那股力量还在，在胸腔里蛰伏着，等待被唤醒。

他闭上眼睛，视野左上角的光屏还在。

```
【全能职业系统 v1.0】
宿主：邱辰
综合评级：D-
声望值：0
当前职业：鉴宝师（D级·未激活）
系统商城：已解锁
主线任务：潘家园淘宝
　└ 剩余时间：约22小时
　└ 任务奖励：激活「鉴宝师」职业，获得声望+100
```

鉴宝师还未激活。

也就是说，他现在还没有真正的鉴宝能力。

但系统在觉醒时给了他一个被动技能——【基础鉴真】D级。30cm范围内，可以鉴别古董真伪和基本断代。

这个技能不需要激活，拿到就是他的。

邱辰睁开眼，嘴角扯了扯。

两百块，加一个30cm的鉴真能力，去潘家园赌一把。

赌赢了，人生翻盘。

赌输了——

他没有想赌输了怎么办。

---

四十分钟后，公交车在一个嘈杂的路口停了下来。

"潘家园到了，开门的乘客请带好随身物品——"

邱辰随着人流下了车。

一股混合着泥土、旧纸、香烛和廉价食物的气味扑面而来。

潘家园地摊区就在眼前。

铁皮搭顶的大棚一眼望不到头，摊位密密麻麻地挤在一起，五颜六色的塑料布上堆着各种旧货——缺口的瓷碗、生锈的铜钱、发黄的古籍、蒙尘的玉佩、林林总总的杂项。摊主们或坐或蹲，守着自己的地盘，眼睛贼亮地盯着每一个路过的人。

"哎，小伙子来看看！老物件！绝对真货！"

"这边这边，刚从农村收上来的，清代的老东西——"

"祖传的玉佩，开过光的，看一眼不收钱——"

吆喝声此起彼伏，夹杂着讨价还价的争吵声和偶尔飘来的脏话。

邱辰没有停留。

他沿着过道慢慢往里走，眼睛看似漫不经心，实际上把两边的摊位都扫了一遍。

这是他从小跟老爸学的——初到一个陌生的地方，不要急着出手，先走一圈，感受一下整个市场的气息，判断哪些摊位是认真做生意的，哪些是专门宰外地人和新手的。

走了大约十分钟，他大致摸清了这一片的情况。

地摊区分为三个档次。

靠近入口的区域，卖的大多是现代工艺品和仿品，专门忽悠不懂行的游客，价格虚高，水最深。

中间区域，有些真货，但假货也不少，需要眼力和经验，摊主相对专业，不好糊弄。

最里面的区域，是老玩家和行内人出没的地方，东西质量最高，价格也最实，但门槛也最高——没有足够的眼力和人脉，很难买到东西。

邱辰想了想，往中间区域走去。

他现在只有两百块，入口那片肯定不行。最里面的那片，摊主都是老江湖，他一个穿着旧T恤的小年轻，拿两百块去砍价，人家连眼皮都懒得抬一下。

中间区域刚刚好。

他继续往里走。

走了大概五十米，他的目光突然定住了。

前方三米外，一个不起眼的小摊位上，摆着一摞杂七杂八的旧物件——几枚铜钱、一只缺了盖的瓷壶、几块玉碎、还有一只用旧报纸垫着的碗。

那只碗。

邱辰的脚步顿了一下。

他下意识往前走了一步，眼睛盯着那只碗，心跳莫名快了几分。

不是因为他看出了什么——他现在还什么都没有看出来。

是一种直觉。

一种从小跟老爸混在古董堆里养出来的直觉。

那个摊位的老头正百无聊赖地坐在小板凳上扇蒲扇，看年纪大概六十出头，皮肤黝黑，满脸皱纹，穿一件洗得发白的蓝色老头衫，脚上蹬着一双塑料拖鞋。

摊位上立着一块手写的小纸牌："老张杂项，概不还价。"

邱辰走过去，在摊位前蹲了下来。

"老板，这碗怎么卖？"

老张抬起眼皮看了他一眼，没精打采地伸出一根手指："一百。"

一百。

就剩两百块的话，一百买一个碗，如果是真的清代官窑——

邱辰还没来得及往下想，他的目光无意间扫过那只碗的边缘，突然，一行蓝色的字迹浮现在他的视野里。

**「物品鉴定中……」**

**「检测范围：30cm」**

**「检测对象：青釉暗刻花卉纹碗」**

**「鉴定结果：赝品（现代高仿）」**

**「建议：谨慎购买」**

邱辰心里一沉。

赝品。

他忍不住又看了一眼那行蓝字——这次不是为了看鉴定结果，而是想确认这串信息是怎么出现的。

系统。

这是他的D级基础鉴真技能在起作用。

但只有30cm范围，而且——

赝品。

他站起身，正准备离开，突然，视野里的蓝色字迹闪了一下。

**「检测到物品附近存在微量异常能量波动」**

**「来源：摊位下方覆盖的旧报纸」**

邱辰愣住了。

摊位下方？

他下意识低头看了一眼老张摊位上的那堆旧报纸——就普普通通的旧报纸，泛黄卷边，堆得乱七八糟，上面还压着一只缺盖的瓷壶。

等等。

邱辰的目光落在那摞报纸最下面的一张上。

那是一张《江城晚报》，日期是二十年前的，头版是某位领导视察的新闻。报纸的边角已经磨损了，但被压在最底下，保存得还算完整。

他的心跳又加快了一分。

二十年。

如果这张报纸下面还压着别的东西——

邱辰重新蹲下来，装作随意地翻了翻摊位上的铜钱和玉碎，然后把手伸向了那摞报纸。

"老板，这报纸卖不卖？"

老张眯了眯眼："那破玩意儿你要它干嘛？"

"我是学新闻的，"邱辰随口扯了个谎，"想找点老报纸当素材。"

老张嗤笑一声："那玩意儿一毛钱一张，你要就拿走。"

"那您这报纸堆里，还藏着什么好东西没有？"邱辰没有动，老老实实地问。

老张的动作顿了一下。

他重新打量了邱辰一眼——这次看得仔细了些，目光在他脸上停留了两三秒。

"你小子眼睛挺毒啊。"老张把蒲扇往旁边一搁，"不过你要的东西，不在这摊子上。"

他从屁股底下摸出一个脏兮兮的布袋子，拉开袋口，往邱辰面前一倒。

哗啦啦，十几件零碎物件滚了出来——有铜钱、有玉佩、有印章、有几枚不知道什么年代的硬币，还有一只用草绳缠着的小瓷瓶。

老张指了指那只小瓷瓶："这个。"

邱辰的呼吸一窒。

那只瓷瓶只有巴掌大小，通体白釉，釉色莹润，瓶身修长，线条流畅优美。瓶口用蜡封着，瓶底有一个豆粒大小的款识，因为年代久远已经有些模糊了。

他伸手想拿，又缩了回来。

——他不能确定这瓶子是不是真的。

30cm范围，只有30cm。

他小心翼翼地把身体凑近了一点，手伸向那只瓷瓶。

就在他的手指即将触碰到瓷瓶的一瞬间，蓝色的字迹再次浮现：

**「物品鉴定中……」**

**「检测范围：30cm」**

**「检测对象：青釉缠枝莲纹瓶」**

**「年代：清·雍正年间」**

**「品级：民窑精品」**

**「估值：人民币8-12万元」**

**「备注：保存完好，具有一定收藏价值」**

**「任务进度：潘家园淘宝——已发现可交易目标」**

邱辰的脑子嗡的一声。

雍正年间。

民窑精品。

八到十二万。

他深吸一口气，把手缩了回来。

"老板，这瓶子怎么卖？"

老张的眼神变得复杂了。

他盯着邱辰看了好几秒，然后慢悠悠地开口："小伙子，你是行里人？"

"不算，"邱辰尽量让自己的声音听起来平静，"就是跟着家里长辈学了一点。"

"学了一点？"老张冷哼一声，"你刚才在我摊子前站了多久，看了几个东西，一眼就盯上了这张报纸——普通买家谁会注意这个？"

邱辰没有说话。

老张叹了口气："算了，不问了。这瓶子，我不打算卖。"

"价格好商量。"

"不是价格的问题，"老张摇摇头，"这东西我收上来的时候，就觉得不是普通货色，一直没舍得出手。本想等个识货的主儿，一口价出个大数目——现在看来，你小子是个行家，但我不想卖给行家。"

"为什么？"

"行家买回去是赚钱的，"老张的眼神里闪过一丝说不清的东西，"我想卖给真正喜欢的人。"

邱辰沉默了几秒。

"我身上只有两百块。"他坦白地说。

老张愣了一下，然后笑了。

"两百块？"他摇摇头，"你小子倒实诚。两百块连这瓶子上的蜡封都买不起。"

"我知道。"邱辰说，"但我可以分期。"

"分期？"

"您开个价，这瓶子多少钱出手，"邱辰说，"我先付两百定金，剩下的每个月还您一笔，多少都行，直到还清为止。"

老张的笑容凝固在脸上。

他盯着邱辰看了很久，目光从他的脸上移到他身上洗得发旧的T恤，再移到他攥紧裤缝的手指。

"你有什么非买不可的理由？"

邱辰想了想，说："我欠了三十二万。"

老张的眼睛微微眯了起来。

"还有别的原因。"他继续说，"我爸以前也是干这行的，五年前失踪了，留下了一堆谜团。这瓶子……是我第一件凭自己的眼睛挑出来的东西。我想留着，当个念想。"

老张沉默了。

过了好一会儿，他从布袋子里摸出那只手掌大小的瓷瓶，在手里掂了掂，然后递向邱辰。

"一千。"

邱辰愣住了。

"一千块，爱要不要。"老张把瓶子往他面前一递，"定金两百，余款八百，下个月还清。"

"您——"

"少废话，"老张挥了挥手，"字据我就不写了，老头子信你一回。"

邱辰接过瓶子的时候，手指有些发抖。

一千块。

原价八到十二万的东西，一千块。

他的脑子里闪过无数个念头，但最终什么都没说，只是郑重地把瓶子捧在手里，然后从贴袋里掏出那两百块皱巴巴的纸币，一张一张地数出来，递到老张手里。

"下个月，八百。"他说。

老张收下钱，点了点头："去吧，别让我失望。"

邱辰捧着那只缠枝莲纹瓶，在地摊区嘈杂的人群中站了好一会儿。

直到视野左上角的蓝色光屏弹出一行新的文字：

**「主线任务完成：潘家园淘宝」**

**「任务评价：A」**

**「奖励发放中……」**

**「奖励一：激活「鉴宝师」职业，当前等级：D级」**

**「奖励二：声望+100（当前声望：100）」**

**「奖励三：现金+800元（老张优惠）」**

**「奖励四：解锁每日任务系统」**

**「系统提示：恭喜宿主完成第一个主线任务！请再接再厉。」**

他低下头，看着怀里的瓷瓶。

午后的阳光从铁皮棚顶的缝隙里漏下来，落在白釉上，泛起一层温润的光泽。

他笑了。

第一次，自父亲失踪以来，他真正地笑了。

（本章完）
